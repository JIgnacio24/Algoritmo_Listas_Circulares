# Algoritmo de Listas Circulares

Proyecto desarrollado en C++ enfocado en la implementación y uso de listas circulares para el manejo y procesamiento de datos cargados desde archivos `.csv`.

## Descripción

Este proyecto pone en práctica estructuras de datos dinámicas, específicamente listas circulares, para almacenar, recorrer y manipular información proveniente de archivos externos.  
El enfoque principal está en aplicar conceptos de algoritmos y estructuras de datos a un caso práctico basado en conjuntos de datos.

## Tecnologías utilizadas

- C++
- Archivos CSV
- Dev-C++ / compiladores compatibles con C++
